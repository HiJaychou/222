# iwantrun VPN Web Manager MVP

轻量级 sing-box 多协议 Web 管理面板第一版骨架。

## 页面结构

- 首页：CPU / 内存 / 硬盘、当前节点状态、网络流量、防火墙提醒、快捷操作
- 协议管理：协议列表、开启 / 关闭 / 重启、节点链接、二维码、更新 sing-box 内核入口
- 用户管理：用户列表、添加 / 删除、查看链接 / 二维码
- 防火墙管理：查看开放端口、添加端口、删除端口、VPS 后台安全组提醒
- 日志管理：sing-box 日志、系统日志、复制完整诊断信息
- 系统设置：修改管理员密码、修改 Web 面板端口、卸载 Web 面板、卸载所有服务

## 安装

```bash
chmod +x install-webui.sh
sudo bash install-webui.sh
```

安装完成后会显示 Web 面板端口、管理员账号和随机密码。

## 说明

这是第一版 WebUI 骨架，已经包含页面、登录、SQLite、系统资源、防火墙端口、日志读取、systemd 服务控制的基础逻辑。协议安装、多协议独立配置写入、用户链接/二维码联动，需要在下一版继续接入。
